# Projet GEII

##  A propos

Le département Génie Électrique et Informatique Industrielle (GEII) de l&#39;IUT souhaite se
doter d&#39;un site web institutionnel moderne afin de centraliser l&#39;ensemble de ses
informations à destination de ses différents publics : futurs étudiants, entreprises
partenaires, étudiants inscrits et enseignants.
Ce site viendra remplacer ou compléter l&#39;offre de communication actuelle et constitue le
point d&#39;entrée numérique officiel du département.

Ce projet n'est pas un projet officiel pour le GEII, mais nous feront tout comme.

## Objectifs du site

Le but de ce site sera de :
• Valoriser l&#39;offre de formation du département GEII (DUT GEII, Licences
Professionnelles).
• Informer les entreprises sur les modalités de partenariat (alternance, projets
tuteurés).
• Offrir aux étudiants inscrits un espace personnel sécurisé (notes, emploi du
temps, supports).
• Permettre aux enseignants de gérer leurs ressources pédagogiques (cours, notes,
emploi du temps).
• Garantir une administration centralisée du contenu par le personnel du
département.
