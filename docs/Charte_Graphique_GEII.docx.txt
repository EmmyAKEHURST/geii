1. Identité visuelle
1.1 Analyse du logo
Le logo du département GEII est construit autour d'une forme en lemniscate (∞) stylisée évoquant à la fois le mouvement continu, l'électricité et les boucles logiques. Il se compose de deux éléments chromatiques distincts :
* Un tracé jaune vif (#FFD700) représentant l'énergie, le dynamisme et l'innovation.
* Un tracé noir (#1A1A1A) apportant rigueur, lisibilité et ancrage institutionnel.
Cette dualité jaune/noir constitue le socle de toute la charte graphique du site.


1.2 Règles d'utilisation du logo
Règle
	Détails
	Zone de protection
	Marge minimum = hauteur de la lettre « G » du logo tout autour
	Taille minimale
	80 px de large sur écran, 25 mm à l'impression
	Fond autorisé
	Blanc (#FFFFFF) ou Noir (#1A1A1A)
	Fond interdit
	Fond coloré saturé, fond image chargée, fond jaune
	Déformation
	Strictement interdite (proportions fixes)
	Recolorisation
	Interdite — uniquement jaune/noir ou version monochrome noire

4.5 Formulaires
Tous les formulaires (connexion, dépôt de fichier, ajout d'offre) suivent ces règles :
* Labels au-dessus des champs (form-label), jamais inline.
* Champs avec focus coloré en jaune via --bs-focus-ring-color : rgba(255,215,0,0.3).
* Messages de validation via les classes is-valid / is-invalid Bootstrap.
* Bouton de soumission : btn btn-warning fw-semibold w-100.
:root { --bs-focus-ring-color: rgba(255, 215, 0, 0.35); }
.form-control:focus { border-color: #FFD700; box-shadow: 0 0 0 0.25rem var(--bs-focus-ring-color); }

6.2 Visuels et illustrations
* Les photos utilisées sur le site (hero, lieu de formation) doivent être au format WebP pour les performances.
* Résolution minimale recommandée : 1920 × 600 px pour les bandeaux hero.
* Toute image doit posséder un attribut alt (texte alternative, accessibilité WCAG 2.1).
* Les images de personnes réelles (étudiants, enseignants) nécessitent un droit à l'image signé.
* Éviter les visuels génériques de stock ; privilégier des photos authentiques du département.

7. Accessibilité et contraste
7.1 Ratios de contraste (WCAG 2.1 AA)
Le jaune (#FFD700) sur fond blanc (#FFFFFF) présente un ratio de contraste de 1.07:1 — insuffisant pour du texte. Il ne doit donc jamais être utilisé comme couleur de texte sur fond blanc.

7.2 Bonnes pratiques Bootstrap
* Utiliser les attributs aria-label et aria-expanded sur les éléments interactifs (menus, modales).
* Les dropdowns Bootstrap incluent automatiquement les rôles ARIA.
* Tester le focus clavier sur toute la navigation (:focus-visible stylisé avec outline jaune).
:focus-visible { outline: 3px solid #FFD700; outline-offset: 2px; }
* Utiliser des balises HTML sémantiques : <nav>, <main>, <section>, <aside>, <footer>.
* Vérifier la charte avec l'outil Lighthouse (Google Chrome DevTools) — score Accessibilité ≥ 90.
